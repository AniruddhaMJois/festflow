## 🌌 Overview
**FestFlow** is an advanced, real-time command center designed for large-scale event management and safety monitoring. Developed by **Team Ignovators**, it provides a high-tech interface for tracking hardware nodes and vital statistics in real-time.

The dashboard features a stunning **Cyberpunk/Circuit-themed UI**, utilizing the **WebSerial API** to bridge the gap between hardware and software seamlessly.

## ✨ Core Features
- **⚡ Neural Link (WebSerial)**: Direct real-time data streaming from monitoring hardware.
- **🗺️ Sector Localization**: Geospatial tracking across multiple zones (HQ, Alpha, Beta, Delta).
- **🚨 Panic Monitoring**: Automated alerts for critical vitals (e.g., HR > 160 BPM).
- **📟 Terminal Stream**: Live serial data debugging and system logs.
- **🎨 Glassmorphic Design**: Interactive circuit backgrounds with dynamic data pulses.

## 🛠️ Technology Stack
- **Frontend**: HTML5, Vanilla CSS3 (Custom Design System), JavaScript (ES6+)
- **Maps**: [Leaflet.js](https://leafletjs.com/)
- **Hardware Integration**: WebSerial API
- **Typography**: Orbitron, Share Tech Mono, Rajdhani

## 🚀 Quick Start
1. **Clone the Repo**:
   ```bash
   git clone https://github.com/AniruddhaMJois/festflow.git
   ```
2. **Launch**:
   Open `index.html` in any modern web browser.
3. **Connect**:
   Authenticate through the portal and use the **LINK_HARDWARE** button to begin monitoring.
